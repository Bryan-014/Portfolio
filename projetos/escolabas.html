Escola Base

Concluido

Descrição: O Escola Base é um sistema básico de gestão escolar, contendo funcionalidades para administração simplificada. O Escola Base foi desenvolvido como projeto final da disciplina de Orientação a Objetos de ADS.

Funcionalidades Desenvolvidas:
  • Controle de Professores e Alunos
  • Controle de Disciplinas
  • Controle de Matriculas
Técnologias que utilizei:
HTML
CSS
Bootstrap
Javalin - Java
MySql

Detalhamento:

Listas

A página inicial de cada módulo é uma lista contendo as informações báscias de cada Disciplina, Professor ou Aluno. Além de contar com as ações que podem ser realizadas:

  • Visualizar
  • Editar
  • Deletar

Visualização

As telas de visualização permitem a leitura dos dados detalhados de Disciplinas, Professores e Alunos.

Além disso, na tela de visualização de disciplinas, é possivel visualizar os alunos matriculados na disciplina, além de permitir a matrticula de novos alunos e desmatricula.

Edição

As telas de edição permitem a atualização de registros de Disciplinas, Professores, Alunos e Escola.

Cadastro

As telas de cadastro permitem a inclusão de novos registros incluindo todas informações de Disciplinas, Professores e Alunos.
