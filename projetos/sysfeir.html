Sys Feiras

Em Desenvolvimento

Descrição: O Sys Feiras é um sistema de controle de vendas desenvolvido para Ambulantes e Feirantes.

Funcionalidades Desenvolvidas:
  • Controle de Vendas
  • Geração de Ralatórios
  • Sistema de Filas(Em Análise)
  • Controle de Estoque(Em Análise)
  • Plataforma para Clientes(Em Análise)
Técnologias utilizadas:
HTML
CSS
JavaScript
Bootstrap
Java - Javalin
Jquery - Ajax
MySql

Detalhamento:

Login

O sistema é dividido entre Administrador e Vendedor.

  • O administrador tem acesso total às funcionalidades da plataforma, podendo controlar usuários, gerenciar produtos, visualizar relatórios e administrar as vendas;
  • O Vendedor tem acesso ao painel de vendas e à visualização dos relatórios.

Painel de Vendas

O vendedor pode selecionar o produto a ser vendido, mudar a quantidade e adcionar uma observação e o registrar na venda, posteriormente pode editar os produtos registrados.

Ao finalizar a venda o vendedor seleciona os métodos de pagamento realizados e seus respectivos valores recebidos

Relatórios

O administrador pode visualizar relatórios de vendas em tempo real para acompanhamento de desempenho.

O administrador pode aplicar filtros para visualizar informações especificas

Controle de Produtos

O Administrador gerencia os produtos disponíveis para venda.

Controle de Usuários

O Administrador tem controle total sobre os usuários cadastrados na plataforma.

Mais Detalhes

O sistema visa simplificar informações, deixando-o com um fluxo mais amigável a feirantes.

Ainda está no planejamento a criação de expanções que devem abranger feirantes do ramo alimentício, além de melhorias na plataforma. Os recursos que planejo implementar são:

  • Sistema de Filas: Sistema para acompanhar a lista de produtos a serem produzidos;
  • Controle de Estoque: Atualização do módulo de produtos vendidos para um sistema de controle de estoque completo;
  • Plataforma para clientes: cardápio virtual, localização em tempo real de feirantes, sistema de delivery e sistema de acompanhamento de pedido online;
  • Atualização do módulo de relatórios: atualização dos filtros, geração de gráficos;
  • Atualização da plataforma: adição dos módulos de notificações, configurações e personalização.
