Senac Saúde

Concluido

Descrição: O Senac Saúde foi o projeto integrador e trabalho final do curso técnico no Senac, o projeto focou no trabalho em equipe e desenvolvimento de uma aplicação hospitalar.

Funcionalidades Desenvolvidas:
  • Sistema de Login
  • Sistema de Notificações
  • Controle de Usuários
  • Controle de Estoque
  • Controle de Protuários
Técnologias utilizadas:
HTML
CSS
Bootstrap
PHP
JavaScript
Jquery - Ajax
MySql

Detalhamento:

Sistema de Login

O login da plataforma é feito de acordo com a categoria do usuário, além também de ser possivel selecionar a unidade que se deseja trabalhar.

Página inicial/Notificações

A página inicial contém links rápidos para todos os módulos do sistema.

Além disso, as notificações podem ser visualizadas em qualquer parte do sistema.

Recepção

Onde tem a lista de paciêntes e formulário para recepção de novos paciêntes.

Na recepção se inicia a criação dos prontuários.

Triagem

Lista os pacientes a serem atendidos na triagem.

Atualiza os prontuários agora com os dados da triagen.

Consulta

Lista os pacientes a serem atendidos.

Visualiza os dados do paciente agora.

Atualiza os prontuários agora com os dados da consulta

Histórico

Visualização completa dos prontuários.

Estoque

Controle geral dos insumos do hospital com detalhes de validade.

Controle de Usuários

Controla os usuários cadastrados na plataforma, além de controlar os módulos que eles têm acesso.

Mais Detalhes

Neste projeto, fiquei mais responsável pelo back-end, porém participei ativamente no front-end também.
